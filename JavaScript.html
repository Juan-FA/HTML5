Historia de JavaScript

Versiones de JavaScript

Versión Año Características
ES1 1997 Primera edición
ES2 1998 Cambios editoriales
ES3 1999 Agregado:
  • Expresiones regulares
  • Try/catch
  • Switch
  • Do-while
ES4 No lanzada Ninguno
ES5 2009 Agregado:
  • "Modo estricto"
  • Soporte para JSON
  • String.trim()
  • Array.isArray()
  • Métodos de iteración para Arrays
  • Permite comas finales en literales de objetos
ES6 (ECMAScript 2015) 2015 Agregado:
  • Let y const
  • Valores predeterminados para parámetros
  • Array.find()
  • Array.findIndex()
ES7 (ECMAScript 2016) 2016 Agregado:
  • Operador exponencial (**)
  • Array.includes()
ES8 (ECMAScript 2017) 2017 Agregado:
  • Relleno de cadenas
  • Object.entries()
  • Object.values()
  • Funciones asíncronas
  • Memoria compartida
  • Permite comas finales en parámetros de funciones
ES9 (ECMAScript 2018) 2018 Agregado:
  • Propiedades rest/spread
  • Iteración asíncrona
  • Promise.finally()
  • Adiciones a RegExp
ES10 (ECMAScript 2019) 2019 Agregado:
  • String.trimStart()
  • String.trimEnd()
  • Array.flat()
  • Object.fromEntries
  • Enlace opcional en catch
ES11 (ECMAScript 2020) 2020 Agregado:
  • El operador de fusión nula (??)
